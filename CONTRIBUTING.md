# Contributing

Contributions are welcome, especially for synchronization validation,
calibration reproducibility, ROS 2 integration, odometry baselines, online
radar processing, and deployment diagnostics.

Please:

1. open an issue describing the hardware and software environment;
2. avoid committing raw recordings, serial numbers, credentials, or private IPs;
3. document coordinate frames, transform direction, units, and timestamp source;
4. add a minimal reproduction and validation evidence for hardware changes;
5. preserve upstream copyright and license notices.
